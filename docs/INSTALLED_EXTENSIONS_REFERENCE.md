## üì¶ Installation Commands

To install all extensions via command line, use:

```bash
# Code Quality & Linting
cursor --install-extension usernamehw.errorlens
cursor --install-extension dbaeumer.vscode-eslint
cursor --install-extension esbenp.prettier-vscode
cursor --install-extension sonarsource.sonarlint-vscode
cursor --install-extension charliermarsh.ruff

# Python
cursor --install-extension ms-python.python
cursor --install-extension ms-python.vscode-pylance
cursor --install-extension ms-python.black-formatter
cursor --install-extension njpwerner.autodocstring
cursor --install-extension littlefoxteam.vscode-python-test-adapter

# Git
cursor --install-extension eamodio.gitlens
cursor --install-extension donjayamanne.githistory
cursor --install-extension github.vscode-pull-request-github

# Database
cursor --install-extension qwtel.sqlite-viewer
cursor --install-extension mtxr.sqltools

# Testing
cursor --install-extension firsttris.vscode-jest-runner
cursor --install-extension ryanluker.vscode-coverage-gutters
cursor --install-extension ms-playwright.playwright

# Documentation
cursor --install-extension yzhang.markdown-all-in-one
cursor --install-extension shd101wyy.markdown-preview-enhanced
cursor --install-extension davidanson.vscode-markdownlint

# Docker
cursor --install-extension ms-azuretools.vscode-docker
cursor --install-extension ms-kubernetes-tools.vscode-kubernetes-tools

# Productivity
cursor --install-extension aaron-bond.better-comments
cursor --install-extension gruntfuggly.todo-tree
cursor --install-extension alefragnani.project-manager
cursor --install-extension wakatime.vscode-wakatime
```

**Note:** Replace `cursor` with `code` if using VS Code instead of Cursor.
