# UI Screenshots Guide

## Overview

This guide provides instructions for capturing professional screenshots of the CryptoOrchestrator platform. Screenshots should showcase key features, modern UI design, and professional polish.

## Screenshot Requirements

### Technical Specifications

- **Format**: PNG or WebP
- **Resolution**: Minimum 1920x1080, preferred 2560x1440 or higher
- **Aspect Ratio**: 16:9 or 4:3
- **File Size**: Optimized (under 500KB when possible)
- **Naming**: Descriptive, kebab-case (e.g., `dashboard-overview.png`)

### Content Guidelines

- ✅ Clean, uncluttered interfaces
- ✅ Realistic sample data
- ✅ Professional color scheme
- ✅ Clear feature visibility
- ✅ Consistent branding

## Screenshot Best Practices

### Before Capturing

1. **Clean UI**: Remove test data, clear notifications
2. **Realistic Data**: Use believable sample data
3. **Consistent Theme**: Use default/production theme
4. **Browser**: Use Chrome/Edge with clean state
5. **Window Size**: Full screen or consistent size

### During Capture

1. **Focus**: Highlight key features clearly
2. **Composition**: Center important elements
3. **Overlays**: Add helpful annotations if needed
4. **Consistency**: Same style across all screenshots

### After Capture

1. **Edit**: Crop, resize, optimize
2. **Annotate**: Add arrows, highlights if needed
3. **Optimize**: Compress for web use
4. **Organize**: Store in `/screenshots` directory

## Demo Video Script

### Video Structure (2-3 minutes)

1. **Introduction (0:00-0:15)**
   - Logo and title card
   - Brief platform overview

2. **Dashboard Tour (0:15-0:45)**
   - Dashboard overview
   - Portfolio summary
   - Quick navigation

3. **Key Features (0:45-1:30)**
   - Creating a trading bot
   - Building a strategy
   - Running a backtest

4. **Advanced Features (1:30-2:15)**
   - AI Copilot interaction
   - Risk management
   - Multi-exchange trading

5. **Conclusion (2:15-2:30)**
   - Platform summary
   - Call-to-action
   - Contact information

### Video Specifications

- **Duration**: 2-3 minutes
- **Resolution**: 1920x1080 (Full HD)
- **Frame Rate**: 30fps
- **Format**: MP4 (H.264)
- **Audio**: Clear narration with background music (optional)
- **Subtitles**: Optional (recommended for accessibility)

### Recording Tools

- **Screen Recording**: OBS Studio, Camtasia, ScreenFlow
- **Editing**: DaVinci Resolve, Adobe Premiere, Final Cut Pro
- **Audio**: Clear microphone, audio editing software

## Notes

- Screenshots and videos should reflect the current production UI
- Update when UI changes significantly
- Maintain consistent style and branding
- Consider accessibility (alt text, captions)
