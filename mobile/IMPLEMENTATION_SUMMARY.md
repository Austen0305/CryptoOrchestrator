Edit `mobile/.env`:
```env

API_BASE_URL=http://localhost:8000

API_BASE_URL=http://192.168.1.X:8000
```

```bash
# Start backend (in project root)
npm run dev:fastapi

# Start mobile (in mobile/)
cd mobile
npm start

# In another terminal:
npm run ios     # For iOS
# OR
npm run android # For Android
```
